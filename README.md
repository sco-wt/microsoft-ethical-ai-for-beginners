# Pathway: Responsible AI

This folder turns Microsoft's "Ethical and Responsible AI" lesson into a short practical learning pathway in English.

Live site, once GitHub Pages is enabled:
[https://sco-wt.github.io/microsoft-ethical-ai-for-beginners/](https://sco-wt.github.io/microsoft-ethical-ai-for-beginners/)

## Start Here

- Open [the home page](index.html) for the GitHub Pages entry point.
- Open [lesson 0001](lessons/0001-transformer-les-principes-en-audit.html) to learn how to turn principles into audit questions.
- Keep the [reference sheet](reference/principes-ia-responsable.html) nearby when you analyze a use case.
- Check [RESOURCES.md](RESOURCES.md) for the trusted sources used throughout the pathway.

## Goal

Understand responsible AI principles, then learn how to apply them to real projects: identify risks, choose safeguards, and document who remains accountable.
